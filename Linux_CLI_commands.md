> # To Displat the count of lines/word/byte.

It will Display lines words bytes

```
wc file_name
```

It will Display only Lines

```
wc -l file_name
```

It will Display only Words

```
wc -w file_name
```

It will Display only Bytes

```
wc -c file_name
```
